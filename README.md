Kemudian buka notebook yang diinginkan dari browser.

## Kesimpulan Umum Perjalanan Belajar (Pertemuan 1–7)

Perjalanan tujuh pertemuan ini membangun pemahaman Data Science secara bertahap dan terstruktur. **Pertemuan 1** menjadi titik awal — mempelajari sintaks Python dasar yang menjadi fondasi semua pekerjaan selanjutnya. **Pertemuan 2** memperkenalkan Pandas sebagai alat utama manipulasi data, dengan praktik langsung menggunakan dataset Titanic melalui operasi *groupby* dan agregasi.

**Pertemuan 3** mengajarkan realita paling penting dalam Data Science: data di dunia nyata selalu kotor. Proses *data cleaning* — menghapus duplikat, menormalisasi string, mengimputasi nilai yang hilang, dan menangani outlier — adalah langkah wajib sebelum analisis apapun. **Pertemuan 4 dan 5** memperdalam kemampuan eksplorasi dan komunikasi data: EDA dengan statistik deskriptif dan distribusi, diikuti visualisasi yang efektif menggunakan Matplotlib dan Seaborn untuk mengungkap pola tersembunyi dalam data.

**Pertemuan 6** menjembatani analisis menuju pemodelan — *feature engineering* seperti one-hot encoding dan normalisasi adalah langkah krusial agar model Machine Learning dapat belajar dengan baik. Puncaknya, **Pertemuan 7** mengimplementasikan model Regresi Linear pertama: melatih model, memprediksi, dan mengevaluasinya dengan metrik MAE, RMSE, dan R². Secara keseluruhan, perjalanan ini membangun pipeline Data Science end-to-end: dari data mentah → pembersihan → eksplorasi → visualisasi → preprocessing → pemodelan → evaluasi.
